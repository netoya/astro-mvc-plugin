# Astro MVC Plugin

This is a plugin for Astro that adds support for the MVC pattern.

## Installation

```sh
npm install astro-mvc
```

## Usage

```sh
npm run dev
```

## Configuration

```json
{
  "plugins": [
    "astro-mvc"
  ]
}
```
